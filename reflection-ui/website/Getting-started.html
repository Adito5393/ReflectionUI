Getting started

How to run

In order to test the library:
  • - Run the following command from the "reflection-ui/target" directory:
    java -jar reflection-ui-<version>.jar
  • - Enter any class name in the input dialog. Example: java.util.Date
  • - Enjoy!

How to use

In order to generate the ReflectionUI jar file and its dependencies, run the following command from the "reflection-ui" directory:
mvn install dependency:copy-dependencies
