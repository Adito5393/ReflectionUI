Getting started

How to run

In order to test the library:
  • - Run the following command from the "reflection-ui" directory:
    mvn install exec:java
  • - Enter any class name in the input dialog. Example: java.util.Date
  • - Enjoy!

How to use

In order to generate the ReflectionUI jar file and its dependencies, run the following command from the "reflection-ui" directory:
mvn install dependency:copy-dependencies

Then start coding: 


/* The Hello world: */
Object myObject = new Date();
ReflectionUI reflectionUI = new ReflectionUI();
reflectionUI.openObjectFrame(myObject, "Hello world", null);

/* You can open a dialog instead of a frame: */
reflectionUI.openObjectDialog(null, myObject, "dialog instead", null,
		true);

/* You can just create a form and then insert it in any container: */
JOptionPane.showMessageDialog(null,
		reflectionUI.createObjectForm(myObject));
